Object类

getClass()

  • 返回引用中存储的实际对象类型

hashCode()

  • 返回对象的hash值，int类型
  • 一般相同对象返回的哈希值相同

toString()

  • 返回该对象的字符串表示
  • 可以根据需求重写该方法，默认返回：字符串表示@十六进制哈希值
  • 源码：return getClass().getName() + "@" + Integer.toHexString(hashCode());

equals()

  • 比较两个对象是否相同
  • 可以根据需求重写该方法
  • 源码：return (this == obj);

finalize()

  • 当对象被判定为垃圾对象时，由JVM自动调用此方法，我们不用亲自调用该方法
  • 自动回收机制：内存不足的时候系统自动调用。
  • 手动回收机制：使用System.gc();通知JVM回收垃圾

包装类

  • 包装类就是基本数据类型所对应的引用数据类型
  • 包装类的默认值是null

类型转换：装箱拆箱

  • 把栈里面的数据拿到对立面变成对象叫装箱，即把基本类型转换为引用类型，反之为拆箱

parseXXX()

整数缓冲区

  • Java预先创建了256个常用整数包装对象
  • integer1和2是分别创建的两个对象，故不相等
  • integer3,4,5,6是用的自动装箱，其实调用的是函数Integer.valueOf(100)。这个函数内部有一个整数缓冲区，自动帮我们创建好了-128到127共256个整数对象。当所给的数值在此区间内则直接将缓冲区的对象拿过来。即integer3,4所指向的对象都是缓冲区的100故为true
  • 而integer5,6超过了缓冲区范围，则是新创建的两个对象

String类

  • 字符串是常量，创建之后不可改变
  • 字符串字面值存储在字符串池中，可以共享
  • 给字符串变量赋值时没有改变原来的数值，而是开辟新的空间创建新的字符串，改变指向地址。
  • String str = new String();创建的字符串产生两个对象一个在堆中，一个在字符串池中。
方法 描述
equals() 比较两个字符串相等
equalsIgnoreCase() 忽略大小写比较两个字符串相等
compareTo() 返回两个字符串第一个不同的字符在字典表里的差值，若都相同则比长度
substring(begin, [end]) 截取对应位置的字符串
length() 返回字符串长度
charAt(index) 返回某个位置的字符
contains(str) 判断是否包含某个子字符串
toCharArray() 转换为字符数组
indexOf(str, fromIndex) 返回子字符串首次出现的位置
lastIndexOf() 返回子字符串末次出现的位置
trim() 去掉字符串前后的空格
toUpperCase()/toLowerCase() 大小写转换
endsWith(str)/startsWith() 判断是否以str开头或结尾
replace(old,new) 字符/字符串替换
split(str) 根据str拆分,使用“[]”选择多个符号拆分

StringBuffer/StringBuilder类

  • 可变长字符串，效率比String高，节省内存
  • Buffer线程安全，Builder线程不安全
  • Buffer效率较低，Builder效率更高
  • 单线程使用StringBuilder
方法 描述
append() 在后面添加
insert(index, str) 指定位置插入
replace(begin,end,str) 制定位置长度替换
delete(begin,end) 删除指定位置片段

BigDecimal类

  • 精确计算浮点数，double是近似存储精度不足
  • 使用的时候一定要用字符串构造对象
方法 描述
subtract() 减法
add() 加法
multiply() 乘法
divide(val, scale, roundingmode) 除法，scale表示保留位数，roundingmode是近似模式，
除不尽的情况要加后两个参数否则报错

Date类

  • 大部分方法已经过时，被Calendar类所替代
方法 描述
after() 判断是否在一个时间之后，返回boolean
before() 判断是否在一个时间之前，返回boolean
compareTo() 比较时间先后，返回1，0，-1
equals() 比较两个时间是否相同，返回boolean

Calendar类

  • 受保护，无法直接创建对象，通过getInstance()方法创建对象
  • 月份是0-11，记得加1
方法 描述
getInstance() 获取当前时间对象
getTime() Calendar转换为Date对象
getTimeInMillis() 获取毫秒值
get(field) 获取对应字段的时间信息
Calendar.YEAR等
set(field, value) 设置时间
add(field,value) 时间加减
getActualMaximum(field) 获取当前字段的最大值
例如获取月份最大天数

SimpleDateFormat类

  • 文本与日期互相转换，格式化
  • parse方法需要抛出异常，因为有可能输入的字符串不符合规定
  • 日期格式：y年，M月，d日，H24小时，h12小时，m分，s秒
方法 描述
formate(Date) 将Date对象按给的格式格式化为字符串
parse(String) 将字符串按给定的格式解析为Date对象
