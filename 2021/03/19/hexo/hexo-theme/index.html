前言

emmm这是第二遍写了，上次忘记保存然后莫得了

看到这篇文章想必你一定时成功安装好了hexo

还没有？

快去看Hexo + Github 分分钟搭建属于自己的博客

hexo最初的主题可能不是很和你的胃口，想要找到更符合自己审美的主题

下面就来告诉大家如去更换主题

找主题

hexo官方网站提供了很多（334个）主题共大家选择.

打开后就可以看到如下图所示

  • 点击图片到预览网站
  • 点击蓝色名字进入对应的Github项目

其实在知乎，CSDN上搜索hexo主题也有很多有用的文章噢

尽情选择吧！

主题下载

  • 一般很流行的主题都会有自己的官方教程
  • 一般在GitHub的项目下也会有README.md的参考文档，建议阅读
  • 下面以我用的Keep主题为例

1.官方（推荐）

阅读主题的官方网页

或者阅读在GitHub的项目下的README.md参考文档，按照步骤来进行安装配置，如图

2.Clone-命令下载（推荐）

点击选中主题下的蓝色标题，进入其GitHub项目

如图绿色的框框，选SSH，复制代码

回到hexo的根文件夹下右键打开Git Bash

粘贴命令并执行，等待安装完毕

3.直接下载

上图蓝色框点击Download ZIP下载压缩包

解压后放入/themes/keep文件夹下（注意不是/themes/keep/keep也不是/themes）

不管是哪种方式最后都应该是这样（注意路径是一般情况，以官方文档为准）

使用主题

  • 注意一下会出现两个_config.yml文件
  • 一个是hexo的配置文件在根目录下，一个是主题的配置文件在你安装好的主题文件夹下

1.选取

打开hexo的配置文件_config.yml找到theme:

等完成之后，你就可以在本地localhost:4000下看到你的主题了

2.配置

我只挑一些有用的，其他建议参考官方网页或官方README.md文档

keep官网

打开主题的配置文件_config.yml

base_info

请在该配置项里正确填写你网站的基本信息。

  • title 网站标题（将显示在网站头部，如下图）
  • author 作者昵称（将显示在网站底部和文章内容页，如下图）
    • 网站底部
    • 文章内容页
  • url 网站域名填写你的域名或yourname.github.io（将用于在文章内容页的版权信息模块正确拼接文章链接，如下图）

avatar

设置作者头像，将显示在文章内容页，如下图。 你可以使用本地图片（即存储 keep/source/images/ 路径下的图片）或图片外链 URL。
写法参考：

  • 本地图片

  • 图片外链

favicon

设置网站 favicon 图标，你可以使用本地图片（即存储 keep/source/images/ 路径下的图片）或图片外链 URL。

home_article

该配置项用于设置首页文章块底部的标签和分类信息显示。

  • category 文章分类
  • tag 文章标签
  • limit 限制显示的个数
  • 效果图
    • 不开启效果
    • 开启效果
