前言

我来啦，第一篇分享就来教教如何搭建自己的博客吧！就像你看到这样。

之前一直以为搭建一个自己的网站费时费力费钱，但是现在才知道Github，才知道并不用费很多心力在前期准备上，可以投入到内容的创作。

通过Hexo，可以快速构建个人博客，并搭建到 Github 上供他人访问。

重点是免费喔

不管你是一位ITer想搭建技术博客，还是一个普通人只想记录生活，看看总没错！

Hexo 介绍

Hexo是一个快速、简洁且高效的博客框架，具有较完善的体系生态，也是一个正在不断开发完善的框架。

Hexo 会解析 markdown 语法 生成对应的 html 并展示在你的网站上。

把仓库命名为yourname.github.io

yourname最好和你的用户名一致

更换主题

我遇到的问题

1.安装git和node后输入node -v无反应

  • 解决
  • 建议安装目录就在C盘不要修改

2.安装hexo后 输入$ hexo -v报错

  • 解决
  • 我当时是因为node.js 版本过低，从官网下载最新版后成功

3.部署时失败

  • 解决
  • 网络问题，有些地方访问不了GitHub，建议翻墙
  • 目录下.deploy_git有问题，建议删除再hexo clean|hexo g|hexo d
