1.注释

  • 书写注释是一个非常好的习惯

2.标识符和关键字

  • Java所有的组成部分都需要名字，类名，变量名，方法名都被称为标识符

  • 只能以（A-Z/a-z/$/_）开头，之后可以是（A-Z/a-z/$/下划线/数字）的任意组合

  • 不能用关键字作为标识符

  • 大小写敏感

  • ex: name, $name, _name

3.数据类型

  • 强类型语言，所有变量先定义才能使用

  • Java的数据类型分为

    • 基本类型

      • 整数类型（数字之间可以用下划线分割：1_000_000）

        • byte
        • short
        • int
        • long long num = 10L;

      • 浮点类型

        • float float num = 3.14F;
        • double

      • 字符类型

        • char

      • 字符串

        • String(不是关键字，是类)

      • 布尔类型

        • boolean

    • 引用类型

      • 类
      • 接口
      • 数组

  • 字节： 位（bit)–>字节（byte/B）–> 1024byte = 1KB

  • 进制

  • 浮点数扩展

    • 避免使用浮点型进行比较
    • 避免使用浮点型进行比较
    • 避免使用浮点型进行比较

  • 字符拓展

    • 强制转换可以把字符转换为数字

    • A = 65； a = 97；大小写直接差32

    • 转义字符

      • \n 换行
      • \t 制表符
      • ……

    • 思考

  • 布尔值扩展

  • 类型转换

    • 优先级：byte, short, char-> int-> long-> float-> double

    • 强制转换（高到低）：(类型)变量

    • 自动转换（低到高）

    • 注意

      1. 不能对布尔类型进行转换
      2. 不能把对象类型转化为不相干的类型
      3. 强制转换存在内存溢出和精度问题

4.变量

  • 作用域

    • 类变量：有static修饰

    • 实例变量：类里方法外，从属于对象

    • 局部变量：只在方法里面有用

  • 常量：用final修饰，不可更改

  • 变量命名规范

    1. 见名知意
    2. 变量：首字母小写+驼峰：lastName
    3. 常量：大写字母+下划线分割：MAX_VALUE
    4. 类名：首字母大写+驼峰：Hello, HelloWorld
    5. 方法名：首字母小写+驼峰：runApp()
