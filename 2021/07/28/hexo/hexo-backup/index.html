  • 分享

写在前面

好久没有写blog了，中间也发生了很多事情，发现当时心血来潮买的mini主机并没有经常用到，而且虚拟机也是个好东西，然后就把它卖了。

然而我忘了我有3篇关于deepin的文章是在那个电脑上写的，我没有存回来。

刚刚发现的时候顿时就慌了

不过好在自己的网站上还有，而且可以复制下来，直接就是markdown的语法，很快恢复了三篇文章。

经此一事，我决定直接把hexo的文件全部备份到github上。

步骤

  • 在github上创建新的仓库比如hexo-backup

  • 在本地的hexo文件夹下打开git bush

  • git init
#初始化
git remote add origin https://github.com/yourname/hexo-backup.git
#添加远程仓库
git add *
git commit -m "backup yy/mm/dd"
git push origin master
#push到远程仓库

后记

我是新建了一个仓库，也可以在原来的仓库新建分支

我还是小白，不是很懂，搞了半天没有成功，而且我也怕影响到博客的静态页面
